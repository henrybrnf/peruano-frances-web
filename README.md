# Sistema Inteligente ML — Detección de Riesgo de Deserción Escolar

Sitio web del proyecto universitario desarrollado para la asignatura **Formulación y Evaluación de Proyectos TI (ISR0832)** de la Universidad Nacional Tecnológica de Lima Sur — **UNTELS**.

## Sobre el Proyecto

Formulación y evaluación de un sistema inteligente basado en Machine Learning para la **detección temprana del riesgo de deserción escolar** en el Colegio Particular **I.E.P. Peruano Francés**, Villa el Salvador, Lima — Perú.

## Contenido del Sitio Web

El sitio presenta los avances de la primera entrega del proyecto:

- **Problema** — Diagnóstico de la deserción escolar en la IE Peruano Francés
- **Institución** — Datos generales verificados en visita de campo
- **Visita al Colegio** — Galería fotográfica de la visita (Abril 2026)
- **Entrevista** — Ficha, hallazgos clave y conclusiones del instrumento aplicado
- **Análisis de Stakeholders** — Registro, matriz de poder/interés, expectativas y plan de comunicación
- **Factores Ambientales** — Internos y externos de la organización
- **Activos de la Organización** — Recursos disponibles para el proyecto
- **Propuesta de Solución** — Sistema ML con 5 módulos, viabilidad y análisis de riesgos
- **Equipo** — Integrantes del grupo

## Stack Tecnológico

- [Next.js 14](https://nextjs.org/) — Framework React con App Router
- [Tailwind CSS](https://tailwindcss.com/) — Estilos utilitarios
- [TypeScript](https://www.typescriptlang.org/) — Tipado estático
- Desplegado en [Vercel](https://vercel.com/)

## Ejecutar Localmente

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) en el navegador.

## Deploy

El proyecto está desplegado automáticamente en Vercel desde la rama `main`.
